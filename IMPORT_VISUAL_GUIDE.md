| Element | Color | Purpose |
|---------|-------|---------|
| Import Button | Purple | Distinguishes from upload |
| Modal Overlay | Black 50% | Focus on modal |
| Selected Card | Purple border | Hover feedback |
| Pre-filled Fields | Green bg | Shows auto-fill |
| Success Message | Green | Positive feedback |
| Error Message | Red | Alert feedback |

## 🖱️ Interaction Guide

### Clickable Elements

```
1. "Import from Folder" button
   └─> Opens modal

2. File card in modal
   └─> Pre-fills form, closes modal

3. "Clear and upload file manually"
   └─> Removes pre-selected file

4. "X" button on modal
   └─> Closes modal without selection

5. "Upload Resource" button
   └─> Submits form (import or upload)
```

### Keyboard Shortcuts

```
ESC → Close modal (if clicked outside)
TAB → Navigate form fields
ENTER → Submit form (when in input)
```

## 📊 State Indicators

### Scanning State
```
Button text: "⏳ Scanning Folder..."
Button color: Gray (disabled)
Spinner visible: Yes
```

### Files Found
```
Message: "✅ Found X new file(s) in [path]"
Modal: Opens automatically
```

### File Selected
```
Message: "✅ Pre-filled form with: [title]"
Form: Scrolls into view
Fields: Auto-filled (green indicators)
```

### Importing State
```
Button text: "⏳ Uploading..."
Button color: Gray (disabled)
Spinner visible: Yes
Form: Disabled
```

### Success
```
Message: "✅ Resource imported successfully!"
Form: Reset to empty
List: Auto-refreshed with new resource
```

## 🔍 Visual Indicators Legend

```
✅ Green checkmark   = Auto-filled / Success
⏳ Spinner          = Loading / Processing
📁 Folder icon      = Import related
📤 Upload icon      = Upload related
🏷️ Tag icon         = Resource type
📊 Chart icon       = File size
> Arrow             = Clickable / Selectable
[X] Close button    = Dismiss / Cancel
```
